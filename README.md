# Delivery Route Optimization for E-commerce

## Overview
This project solves a real-world e-commerce logistics problem by integrating multiple algorithmic paradigms:
- **Unit 1:** Recurrence relations (route cost estimation)
- **Unit 2:** Greedy (parcel selection) + DP (time-window feasibility)
- **Unit 3:** Dijkstra (shortest path) + Prim's MST
- **Unit 4:** TSP (brute-force & Held-Karp DP)
